# 📐 GeoCalculator CI

Un sistema modular desarrollado en **.NET 7** para el cálculo de figuras geométricas, diseñado bajo principios de **Clean Code** e **Integración Continua (CI)**.



[Image of geometric shapes formulas for area and perimeter]


## 📝 Descripción
Este proyecto es una biblioteca de clases que permite realizar cálculos matemáticos de diversas formas geométricas. Está estructurado para ser fácilmente escalable y cuenta con una suite completa de pruebas unitarias para garantizar la precisión de los resultados.

## 🛠️ Tecnologías y Herramientas
* **Lenguaje:** C# 11
* **Framework:** .NET 7.0
* **Motor de Pruebas:** xUnit 2.4.2
* **Test Runner:** Microsoft.NET.Test.Sdk
* **Análisis de Cobertura:** Coverlet.collector

## 📂 Estructura del Repositorio
* `GeoApp/`: Proyecto principal con la lógica de los modelos y cálculos.
* `GeoApp.Tests/`: Proyecto de pruebas unitarias que valida cada fórmula matemática.
